### Skills to Know About
- `/plan` — Activates plan mode before making changes. Use this before tackling any non-trivial feature so Claude proposes an approach and you can redirect before code gets written. Used for architecture decisions like the communities roles system and recommendation wizard redesign.
- `/clear` — Clears conversation context. Use this when switching tasks or after a long session to avoid context drift. This is the most-used command on the team — get in the habit of clearing between unrelated tasks.
- `/usage` — Shows your Claude usage stats. Useful for keeping an eye on session counts and which commands you lean on.
- `/model` — Switches the Claude model mid-session. Useful when a task needs more reasoning power (switch to Opus) or you want faster responses for simpler tasks.
